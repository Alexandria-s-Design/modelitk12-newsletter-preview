# The Systems Thinker's Weekly

**Weekly insights on systems modeling, NGSS alignment, and educational innovation from ModelIt! K12**

## About This Newsletter

The Systems Thinker's Weekly brings practical insights from the ModelIt! K12 platform, real research from Cell Collective, and concrete strategies for implementing systems modeling in your classroom.

Published every Sunday at 8 AM.

## Design

This newsletter blog matches the exact design of [ModelItK12.com](https://charlesmartinedd.github.io/modelitk12/) with:

- **Brand Colors**: ModelIt! blue palette (#030C3C, #0F6ACE, #047abe)
- **Typography**: Hind Vadodara font family
- **Layout**: Clean, professional, research-focused
- **Mobile Responsive**: Optimized for all devices

## Technical Stack

- **Static Site Generator**: Jekyll 4.3
- **Hosting**: GitHub Pages
- **Automation**: n8n workflows for weekly publishing
- **Theme**: Custom design matching ModelItK12 branding

## Local Development

```bash
# Install dependencies
bundle install

# Serve locally
bundle exec jekyll serve

# Visit http://localhost:4000/modelitk12-newsletter/
```

## Automated Publishing

This newsletter uses n8n workflows to automatically:
1. Create new newsletter posts every Sunday
2. Generate AI-powered hero images
3. Commit and push to GitHub
4. Trigger GitHub Pages rebuild
5. Send email notifications to subscribers

See `n8n-workflows/` for automation details.

## Content Schedule

- **Week 1**: Platform Spotlights (ModelIt! features, models, tools)
- **Week 2**: Behind the Science (Research team, publications, discoveries)
- **Week 3**: NGSS Standards (Alignment guides, lesson plans, assessments)
- **Week 4**: Teacher Toolkit (Implementation tips, classroom strategies, success stories)
